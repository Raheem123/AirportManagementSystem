# Airport Management System

A desktop-based airport management system built with **Java**, **Java Swing**, and **MySQL**.

## Planned capabilities

- Manage passengers, flights, staff, and baggage
- Store data in a normalized database designed to Third Normal Form (3NF)
- Use database transactions and rollbacks for operations that must remain consistent
- Provide a Java Swing interface for daily airport operations

## Technology stack

- Java
- Java Swing
- MySQL
- Git and GitHub

## Development approach

The project will be built incrementally, with a Git commit after each meaningful milestone.
